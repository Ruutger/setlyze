setlyze.analysis.batch — Batch mode¶

Author:Serrano Pereira
Release:1.0
Date:April 05, 2013

Module Contents¶

Start an analysis in batch mode.

Batch mode for an analysis is started by instantiating its BeginBatch class. In batch mode, the selected analysis is repeated for a selection of species. The user can select multiple species and the analysis is repeated for each species separately and the results are displayed in a summary report.

class setlyze.analysis.batch.Begin[source]¶

Start an analysis in batch mode.

First the user can select which analysis to start, then the analysis is started in batch mode.

on_analysis_selected(sender, analysis)[source]¶

Start the selected analysis analysis in batch mode.
