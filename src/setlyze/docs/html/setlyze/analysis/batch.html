setlyze.analysis.batch — Batch mode¶

Author:Serrano Pereira
Release:0.2
Date:March 08, 2013

This module starts analyses in batch mode.

The selected analysis is repeated for a selection of species. If all species are selected by the user, the analysis is repeated for each species and the results are displayed in a single report.

class setlyze.analysis.batch.Begin[source]¶

Select which analysis to run in batch mode:

  1. Let the user select an analysis.
  2. Run the analysis in batch mode.
on_analysis_selected(sender, analysis)[source]¶

Start the selected analysis in batch mode.
